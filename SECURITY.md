## Contact

If you discover a security vulnerability, please report it to us privately at:

- Email: `skullgamingg31@gmail.com`

We prefer private reports (not public issues). If you cannot send sensitive
information by email, you may instead use GitHub Security Advisories for this
repository.

See also the project's Code of Conduct: `CODE_OF_CONDUCT.md` — reports that
involve harassment or other community-safety concerns may be handled under that
policy as well.

## Acknowledgement

We will acknowledge receipt of a vulnerability report within 72 hours.

## Triage & Timeline

- Critical / High: aim to provide an initial mitigation or patch within 30 days.
- Medium: target 60 days.
- Low: target 90 days.

These are targets and may change based on the complexity of the issue and the
availability of maintainers.

## Reporting Guidance — what to include

Please include as much of the following as you can:

- Affected version(s) and environment
- A clear description of the issue and impact
- Steps to reproduce (minimal PoC preferred)
- Any logs, stack traces, or network captures
- Suggested mitigations (optional)
- Your preferred contact method (and PGP key if you want encrypted replies)

If you provide a PGP public key we will use it to encrypt sensitive replies where
appropriate.

## Scope

This policy applies to code and configuration in this repository and maintained
CI workflows. It does not cover third-party services, external infrastructure,
or hosted services that we do not operate.

## CVE and Public Advisories

We will coordinate assignment of a CVE identifier when appropriate and may
publish advisory details via GitHub Security Advisories once a fix or
mitigation is available.

## Automated Security

We run Dependabot for dependency updates and security alerts. New alerts may
result in automated PRs proposing upgrades; maintainers will review and merge
as appropriate.

## Disclosure Policy

We prefer coordinated disclosure. If a valid vulnerability is not addressed in
90 days and there is no agreed extension, we may disclose publicly so affected
users can take action.
